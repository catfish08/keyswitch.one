# keyswitch.one

keyswitch.one is an open-source single-key keyboard that happens to fit on your keychain. Supporting both Cherry-MX and Matias switches. Get one soon at [keyswitch.one](http://keyswitch.one).

![image of keyswitch.one](http://i.imgur.com/rHafuSy.jpg)

![image of PCB](http://i.imgur.com/SLfwTyD.png)

Original design files for schematic and layout are provided in Diptrace format. Firmware is a Silabs Simpliicty Studio 3.x Project. MIT-licensed.
